Features

  • Dubeolsik (두벌식) keyboard layout
  • Long-press for archaic character variants
  • Automatic syllable composition
  • Support for 16+ archaic Korean letters
  • Progressive Web App (PWA) functionality
  • Offline support
  • Responsive design for all devices
  • Local note storage

Getting Started

To use the full notetaking app:

  1. Go to the main app: Korean Keyboard PWA
  2. Click "New Note" to create your first note
  3. Use the virtual keyboard to type Korean text
  4. Hold down keys to access archaic character variants
  5. Your notes are automatically saved locally
